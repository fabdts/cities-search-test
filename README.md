
# Test recherche de villes




## API Reference

#### Get cities

```http
  GET /cities
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `q` | `string` | **Required**. Your search value |


## Installation

Clone this repository

### Start the front
```bash
  cd front
  yarn install
  yarn start
```

### Start the back
```bash
  cd back
  yarn install
  yarn start:dev
```
